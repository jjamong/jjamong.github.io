    infra

    INFRA

    하드웨어, OS, 미들웨어, 네트워크 등의 시스템 기반

    Hardware 하드웨어

    서버 장비 본체나 데이터를 저장하기 위한 스토리지, 전원 장치 인프라 가장 low-level. CPU, 메모리, 데이터 스토리지 등

    Network

    사용자가 원격으로 접근할 수 있도록 서버를 연결하는 도구들 MAC주소, IP주소 등

    OS

    하드웨어와 네트워크 장비를 제어하기 위한 기본적인 소프트웨어 window, Linux 등

    Middleware 미들웨어

    웹서버, DBMS, 시스템 모니터링 툴등 비즈니스를 처리하는 애플리케이션의 각종 소프트웨어

    시스템 모니터링 System Monitoring

    Zabbix, Datadog, Mackerel 등
