    Hugo

    Hugo

    Static Site Generator

    정보

    사이트 : https://gohugo.io/
    설치 사이트 : https://github.com/gohugoio/hugo/releases

    명령어

    hugo 프로젝트 생성 (project 프로젝트명)

    hugo new site project

    로컬서버 실행 localhost:1313

    hugo server -D

    빌드하여 public폴더에 생성한다. (theme 테마명))

    hugo -t theme
