디렉토리 구조

https://gohugo.io/getting-started/directory-structure/

명령창에서 hugo new site 실행하면 다음 구성으로 디렉토리 구조가 생성된다.

.
├── archetypes
├── config.toml
├── content
├── data
├── layouts
├── static
└── themes
